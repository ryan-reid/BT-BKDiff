# books.txt

### Overview
Controls functionalities of Tomes (Identify, Town Portal) and their scroll interactions.

### Data Fields
* **Name**: Book reference name.
* **ScrollSpellCode / BookSpellCode**: Linked item codes.
* **pSpell**: Item spell function ID.

| Code | Description |
|---|---|
| 1 | Identify |
| 2 | Town Portal |
| ... | (Potions and Skills) |
